README:
The file hamming.py contains the code for the Hamming Code Calculator written by Elijah Verdoorn for CS 241: Hardware Design at St. Olaf College in the fall of 2014.
The project conatins features beyond those of the assingment:

- A more advanced prompt that allows the user to select from 8 or 16 bit Hamming calculation
- implementation of a method for the user to quit at any time
- ability to calculate and return 8 bit hamming values in addition to 16 bit hamming values
